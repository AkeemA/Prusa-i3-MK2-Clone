# MK2 Clone r2 - 3D Printer

In this repository you will find:
 * frame cut drawing (to be cut from 10mm width material, eg. plywood)
 * printed parts needed
 * bill of materials
 * Customized Marlin 1.1.8
 * printing profiles for Slic3r PE
